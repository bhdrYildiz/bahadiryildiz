# Tahterevalli Simülasyonu

Bu proje, kullanıcı tıklamalarıyla ağırlık eklenebilen ve fizik kurallarına göre dengelenen bir tahterevalli simülasyonu geliştirmeyi amaçlıyor. Proje boyunca yalnızca HTML, CSS ve saf JavaScript kullanılacaktır.
